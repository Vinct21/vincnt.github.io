Watch Shop Project

  • Category: Object-Oriented Programming
  • Created Using: Java and Eclipse
  • Project Date: 23 September 2023
  • Github URL: Project Link

Watch Shop Project is project features a Graphical User Interface (GUI) built with JavaFX to facilitate seamless interaction between customers and sellers in an online store environment.

Customer Use Case:
- Sign Up & Log In: Users can create accounts and log in to their profiles.
- Add Products to Cart: Customers can browse and add products to their shopping cart.
- Purchase Products: Complete the checkout process and make purchases.

Seller Use Case:
- Log In: Sellers can securely log into their accounts.
- Manage Brand: Sellers can add, edit, and remove their brand information.
- Manage Products: Sellers have full control to add, update, and delete products listed on the platform.

Backend & Database:
- The project is integrated with a local MySQL database using JDBC to store user data, product information, and transaction records.
- XAMPP is used to host the local server and connect the Java application to the MySQL database.

This project demonstrates proficiency in Java programming, database management, and the implementation of a user-friendly interface with JavaFX.
